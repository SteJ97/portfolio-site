Modify with Stylish

About

With the use of a simple add-on, you can edit any CSS of any site. Below is a tutorial on how to achieve this with an example.

Jump In

Prerequisites

Before you use this add on, I recommend you at least gain some knowledge of CSS.
Here is a quick crash course to gain a bass of CSS info:
Learn CSS in 20 Minutes

Installing Stylish

Head over to Stylish for Chrome and add the extension.

Once this is done, you should see a stylish icon on the top right of your browser

Tutorial

Click on the stylish icon at the top of your browser. Click on the 3 dots to load the style menu and select “Create New Style”.

The following window should show:

Now give the style sheet a name. If you want to style just one specific website or even website page, click on the specify button and input the URL you want to modify.

In order to get the element, you want to restyle, right click on the website page and select” Inspect” or” inspect element”.

Use the select toolto easily select what element, you want to modify.

Once you have selected the element, copy the ID or class name:

Go back to the stylish window and paste the class / ID name into the code window. Add a name for the style too.

Add you CSS edit for the element you want to change.

Sometimes you will have to add !important to the end of the CSS style if it has to override a style already on the site.

Click save and enjoy your new edit.

About the work

More about the work

Project Brief

As a University mini project, i have created a responsive iframe application that shows websites at 5 given resolutions.

This application can help in the development of a site, as each iframe can show a different perspective of the site, ranging from Mobile phone to Desktop Monitor. I personally find this application faster and easier to use than the included toggle device web tool on chrome. Resolutions for each iframe are also shown, so user's know what resolutions they're working with.

Methods

How it works

- The application is relies mostly on the style of the iframe for each resolution.

- When each of the buttons in the navigation bar are clicked, the iframe style is changed, such as the width and height.

- This shows how a site would look on most devices corresponding to the name of the button clicked.

- Now a problem can emerge with this type of application.

- If the resolution of your monitor is lower than the iframes, the iframe will not show 100% in the browser window.

- To combat this, a set of media queries have been created to change the scale of the iframe at certain resolutions.

- This allows for the iframes to show 100% without changing how the site would look.

Languages

HTML was used for the content, such as the search bar and iframe.

CSS was used just to style the navigation bar.

JavaScript allowed for each button to perform they're needed function.

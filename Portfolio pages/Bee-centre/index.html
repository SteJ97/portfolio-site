The Bee Centre

About the work

This site acts as an informative piece of work, aimed towards people concerned or interested on the impact bee's have on the environment.

More about the work

Project Brief

As a Project brief for university, i have designed and created an informative site for 'The Bee Centre', an environmental orientated business focused on bee's.

I was required to create a 5 page site, which each page required a variety of different types of information, including basic paragraph texts, galleries and a usable contact form. The whole colour scheme of the site follows what the client asked for along with a theme type theme.

Methods

Methdology

  • Requirements

    I broke down the brief to create a requirements list. This includes everything from the colours to use, to every piece of content that needs to be included.

  • Analysis

    The requirements list is analysed to get the design stage ready. This allows for me to know every feature that needs to be included in the design(s).

  • Design

    Usually with Photoshop, i will then create a set of draft designs. I then choose the best design and refine it.

  • Code

    I now code the product page by page based on the final design. Visual code is definetly my choice of software here!

  • Testing

    Now it's time to look for bugs! Every feature is tested and i look at every page at as low and high of a resolution as i can. Usually from 250px - 1440px.

Languages

HTML was used for the content, from the paragraphs to the contact form.

CSS allowed for the Bee colour theme the site holds.

JavaScript allowed for the gallery to work and to load the navbar on every page.

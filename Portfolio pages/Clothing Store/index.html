About the work

This site acts as an informative piece of work, aimed towards people concerned or interested on the impact bee's have on the environment.

More about the work

Project Brief

Designing a website that is part of the fasion industry, the largest ecommerce industry around, is a good way of me getting involved with this part of ecommerce website design and such a project has allowed me to gain more UX experience in both design and the use of tools such as Adobe XD.

Methods

Methdology

  • Requirements

    Picking the top 5 biggest clothing sites around allowed for me to create a set of requirements I needed for the project site.

  • Analysis

    Analysing these set of requirements allowed me to gain an idea of how the site needs to function and look.

  • Design

    Using the 5 sites as a type of base, I tried to go reuse the good aspects of the sites along with throwing in my own ideas to hopefully create a more fluid and minimal design. Adobe XD created the wireframes, storyboards and final design.

  • Testing

    Adobe XD allows for a high-fidelity prototype to be created using the final designs. Having a few users test my protype allowed for user feedback that can be put into consideration for this project and for other projects to come.

Tools
